Week 7

Notes and Stuff

  • Functions have properties and methods
    function square(x) { return x*x; }
    square.length << 1
  • call() method can be used to set the value of this inside a function
    sayHello.call(bruce);
    << 'Hello, my name is Bruce'
  • Can make custom properties to Functions
    pickNose.description = 'you have two new boogers';
    << 'you have two new boogers';
  • Can also cache in different values to be retrieved later through Memoization
  • IIFE functions are defined as they are invoked
    (function(){
    const temp = 'World';
    console.log(`Hello ${temp}`);
    })(); << 'Hello World'
  • Can write temporary variables. Useful because variables can't be removed from a scope once declared. Place the variable inside an IIFE to make it temporary
    (()=>{
    const temp = 2;
    })();
  • IIFE code is great for code that doesn't need to be reused or for enclosing code blocks in a private scope
  • A function can call, define, and redefine itself by assigning an anonymous function to a variable with the same name as the function
    function party(){
    console.log('Wow this is amazing!');
    party = function(){
    console.log('Been there, got the T-Shirt');
    }
    }
  • Recursive functions invoke themselves until a certain condition is met.
    function factorial(n) {
    if (n === 0) {
    return 1;
    } else {
    return n * factorial(n - 1);
    }
    }
  • AJAX is a technique that updates web pages without reloading. Allows web pages to communicate asynchronously with a server
