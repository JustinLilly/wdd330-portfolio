Notes and Stuff

  • Computers won't store stuff unless you tell them to. Local Storage is helpful for this.
    • function loadStory()
    { let storyName = document.getElementById('name_input').value let storyHTML = localStorage.getItem(storyName) document.getElementById('story_editor').value = storyHTML }
    function saveStory()
    { let storyName = document.getElementById('name_input').value let storyHTML = document.getElementById('story_editor').value localStorage.setItem(storyName, storyHTML) }
    function displayStory()
    { let storyHTML = document.getElementById('story_editor').value document.getElementById('story_display').innerHTML = storyHTML }
  • User Experience paradigm shifts every few years. Very important to stay with the times.
  • Design for mobile first.
