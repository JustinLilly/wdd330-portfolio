Notes and Stuff

Browser Object Model (BOM):

  • Can learn which browser is being used
  • Dimensions of the screen
  • Which pages were visited prior to the current one
  • Creating popup windows
  • The above and more can be accessed by the 'window' object
  • Global variables are actually part of the global object
  • In the browser environment, the global object is the window object (so you don't have to worry about typing window.x, etc.)

  • Dialogs

  • alert() pauses execution of the program to display a dialog box message
  • confirm() same as alert() but provides a boolean value option
  • prompt() same as alert(), but gives an input field
  • Be careful with dialogs because them stopping the program exectution can mess with things like callback functions

  • The navigator property will give you browser information. The userAgent property will give browser and operating system info
  • location property contains info about the current page url
  • href property returns the full URL as a string
  • protocol property returns the protocol being used
  • host property returns a string of the current domain
  • history property can access info about previously visited pages in the current browser session
